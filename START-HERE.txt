## 🔧 How to Edit

### Using Any Text Editor
1. Right-click `global-defense-command.html`
2. Choose "Open With" → Choose your editor
3. Find the text you want to change
4. Edit it
5. Save (Ctrl+S)
6. Refresh your browser

### Using VS Code (Recommended)
1. Open VS Code
2. File → Open File
3. Choose `global-defense-command.html`
4. Use Find & Replace (Ctrl+H)
5. Make changes
6. Save
7. Refresh browser

## 🆘 Troubleshooting

### App won't open?
- Make sure you double-clicked the `.html` file
- Try a different browser
- Check file wasn't corrupted

### Fonts look weird?
- Needs internet for Google Fonts
- Fallback fonts will be used offline
- Both work fine!

### Changes not showing?
- Save your edit (Ctrl+S)
- Refresh browser (F5)
- Clear cache (Ctrl+Shift+Delete)

### Animations not smooth?
- Update your browser
- Disable browser extensions
- Enable GPU acceleration
